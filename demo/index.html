This page demonstrates the embeddable chatbot widget. Click the chat button in the bottom-right corner.

Demo mode: To connect to a real backend, update the data-api-url attribute in the script tag below to point to your deployed Netlify Function or Cloudflare Worker.

How to Embed

Add this single script tag to any website:

<script
  src="widget.min.js"
  data-api-url="https://your-backend.netlify.app/.netlify/functions/chat"
  data-primary-color="#0f172a"
  data-greeting="Hi! How can I help you today?"
  data-bot-name="Aria"
  data-fallback-phone="(555) 123-4567"
  data-fallback-email="hello@example.com"
></script>
